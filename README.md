# FishingSecure.ru

**FishingSecure** is an educational web platform dedicated to cybersecurity awareness, specifically focusing on preventing phishing attacks among youth and teenagers. 

This project was developed as a **final 10th-grade school project** to demonstrate practical skills in web development and information security. The primary goal is to educate young internet users about the dangers of phishing and provide them with actionable knowledge to protect their personal data online.

**Live Demo:** [View on GitHub Pages](https://code80code10.github.io/fishingsecure.ru/))

---

## Project Overview

Phishing remains one of the most common and effective cyberattacks targeting younger demographics, who are often highly active on social media, gaming platforms, and messaging apps. This website serves as an interactive, easy-to-understand resource to bridge the knowledge gap and promote safe digital habits.

---

## Website Pages & Features

The website is designed with a user-friendly, educational flow and includes the following key sections:

- **Home Page (`index.html`)**  
  The landing page introduces the project's mission, providing a brief, engaging overview of why cybersecurity matters for teenagers and young adults.

- **What is Phishing?**  
  An educational section that breaks down the mechanics of phishing attacks. It explains how attackers use fake emails, social media messages, and counterfeit websites to steal credentials, complete with real-world examples relevant to youth.

- **How to Protect Yourself**  
  A practical guide offering actionable tips for staying safe online. Topics include verifying HTTPS connections, recognizing suspicious URLs, enabling Two-Factor Authentication (2FA), and safely handling unsolicited messages.

- **Interactive Quiz (JavaScript-powered)**  
  An engaging, interactive module that allows users to test their knowledge about real-world phishing scenarios. This gamified element reinforces learning and makes cybersecurity education more accessible and memorable.

---

## Technologies Used

The project is built using foundational web technologies, ensuring fast load times, responsiveness, and broad browser compatibility:

- **HTML5**: For semantic, accessible, and well-structured page content.
- **CSS3**: For modern, responsive, and visually appealing styling.
- **JavaScript (ES6)**: For dynamic UI components and interactive elements, such as the phishing awareness quiz logic.
